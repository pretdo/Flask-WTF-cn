Flask-WTF¶

Flask-WTF 提供了简单地 WTForms 的集成。

当前版本¶

Flask-WTF 的当前版本是 0.9.5。

其它版本的文档请见 Read the Docs。

功能¶

  • 集成 wtforms。
  • 带有 csrf 令牌的安全表单。
  • 全局的 csrf 保护。
  • 支持验证码（Recaptcha）。
  • 与 Flask-Uploads 一起支持文件上传。
  • 国际化集成。
